# Isaac Love

I build tools for organizations that can't afford to build them.

Co-founder and operator of [Peanut Butter Sundays](https://peanutbuttersundays.org), a youth-led LA nonprofit. Incoming USC Computational Neuroscience student, currently on a gap year building civic technology.

**[empty-homes](https://github.com/lllove514/empty-homes)** is the current project: a public map and accountability engine for Philadelphia's 36,713 likely-vacant properties, built with the housing-rights movement Poor People's Army. It fuses four city datasets nobody had joined, entity-resolves 20,000+ owners to surface shell-LLC networks, and puts a citation-verified AI layer on top that cannot invent a property. Python standard library and vanilla JS, no frameworks.
